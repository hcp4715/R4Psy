## 附：PE / δ 与推论一致性规则

PE / δ 计算：

```text
PE = |原文献报告值 - 复现值| / |原文献报告值| × 100%
```

评级规则：

| 评级 | 判断标准 |
|---|---|
| 完全一致 | PE = 0% |
| 次要偏差 / 偏差较小 | 0% < PE < 10% |
| 主要偏差 / 偏差较大 | PE ≥ 10% |
| 舍入偏差 | 差异可由小数位数或四舍五入解释 |
| 无法检验 | 数据缺失、代码缺失、说明不清或无法定位对应结果 |

推论一致性规则：除非原文另有说明，默认 α = .05。原文 p 值和复现 p 值在 α 边界同侧为一致，不同侧为不一致。
